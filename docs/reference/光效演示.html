手电筒原理

手电筒是解谜游戏中最常用的沉浸式光效，它会跟随玩家的鼠标移动。

CSS 径向渐变

使用 radial-gradient 创造一个中心透明，四周纯黑的圆形遮罩。

background: radial-gradient(
  circle 150px at var(--cursor-x) var(--cursor-y),
  transparent 0%,
  rgba(0,0,0,0.7) 80%,
  #000 100%
);

JS 坐标追踪

监听鼠标事件，实时将坐标赋值给 CSS 变量 --cursor-x。

container.addEventListener('mousemove', (e) => {
  layer.style.setProperty(
    '--cursor-x', `${e.clientX}px`
  );
});

鼠标穿透秘籍

一定要在遮罩层加上 pointer-events: none;。否则玩家点击物品时，点到的其实是“光罩”，而不是下面的物品！
